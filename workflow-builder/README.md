# JointJS+: Workflow Builder <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

The Workflow Builder demo app lets you visually design automation workflows suitable for various domains. This general-purpose workflow builder demonstrates how JointJS+ can be used to create flexible visual programming interfaces.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/workflow-builder).

## Available Versions

- [JavaScript](./js/)
- [TypeScript](./ts/)
