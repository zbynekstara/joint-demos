# JointJS+: Collapse / Expand <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

The Collapse / Expand demo utilizes the JointJS+ TreeLayout while providing the user with ability to collapse and expand branches.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/collapsible).
