# JointJS+: Inspector Dynamic Element Type <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

Do you need to use the Inspector to change element types dynamically? The following demo allows users to drag elements from a stencil palette to the JointJS paper, and then dynamically change the element type via the Inspector.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/inspector-dynamic-element-type).

- [JavaScript](./js/)
