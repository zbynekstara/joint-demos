# JointJS+: Comment View <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

Want to enable a special mode where users can only post comments? Allow them to only add or delete comments, but not interfere with the structure of the diagram? Take a look at this demo.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/comment-view).
