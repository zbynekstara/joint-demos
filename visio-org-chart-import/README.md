# Visio Org Chart Import

This demo imports a Microsoft Visio VSDX file containing an organizational chart and converts Visio Shapes to JointJS cells.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/visio-org-chart-import).

## Available Versions

- [JavaScript](./js/)
- [TypeScript](./ts/)
