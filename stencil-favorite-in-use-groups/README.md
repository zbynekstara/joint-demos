# JointJS+: Stencil Favorite & In-Use Groups <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

Explore a demo that introduces a clever feature that allows you or your customers to add shapes to favorites (for easy reuse) and automatically group shapes that are used on paper. Your preferences are persistent, which means that if you refresh your browser, you can freely use previously saved shapes again.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/stencil-favorite-in-use-groups).
