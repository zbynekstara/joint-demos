# JointJS: Scale Option For Tools (JavaScript)

Need to scale element or link tools up or down in an easy manner? Using the scale option available with tools, you can achieve just that. The default level is 1, so just adjust the value to find something which visually suits your application.

## How to download this demo

You can download this demo using our [`@joint/cli` tool](https://www.npmjs.com/package/@joint/cli):

```bash
npx @joint/cli download scale-option-for-tools/js
```

Alternatively, you can get the [copy of the repository](https://github.com/clientIO/joint-demos/archive/refs/heads/main.zip) from GitHub as usual.

## Running the application

Install the dependencies by running:

```bash
npm install
```

And then start the application with:

```bash
npm run dev
```
