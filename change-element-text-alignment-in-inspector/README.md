# JointJS+: Change Element Text Alignment in Inspector <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

How to configure an inspector field to control multiple attributes at once? This demo shows one of the options: to define a custom element which binds a high-level attribute (textAlignment) to multiple presentation attributes (textAnchor, textVerticalAnchor. x, y).

This demo is also available online at [jointjs.com](https://jointjs.com/demos/change-element-text-alignment-in-inspector).
