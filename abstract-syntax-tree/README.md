# JointJS+: AST Visualizer <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

The Abstract Syntax Tree (AST) Visualizer is a great example of a JointJS+ demo that makes life easier for developers. It offers an alternative to the good old AST Explorer and serves as a graphical representation of the source code.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/abstract-syntax-tree).

## Available Versions

- [JavaScript](./js/)
- [TypeScript](./ts/)
