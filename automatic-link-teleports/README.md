# JointJS+: Automatic link teleports <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

As diagrams get more complex, long links can clutter the canvas by crossing over other elements. Teleport links avoid this by replacing the visible path with small markers at each end. Clicking a marker moves the viewport to the other end, so users can navigate between connected shapes without the visual noise of a long link. Markers appear automatically for long links, while short links stay visible. This example shows how to implement teleport links in a diagram.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/automatic-link-teleports).
