# JointJS+: Dialog Generator <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

This demo demonstrates how to create a question and answer diagram. Users can create a dialog, then preview the result in an interactive manner. This type of flowchart can be used to convey a wide range of information.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/dialog-generator).

## Available Versions

- [JavaScript](./js/)
