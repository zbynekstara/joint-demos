# JointJS+: Presentation Mode <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

Do you want to present a diagram to your users element by element? Do you want them to navigate through the diagram using the presentation toolbar? Here's a demo that does just that.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/presentation-mode).
