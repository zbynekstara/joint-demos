# JointJS+: Timeline <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

This timeline demo not only takes you back in time and highlights key milestones for OpenAI and Anthropic, but also showcases advanced diagramming features—such as automatic layout and drag-and-drop—available in JointJS+.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/timeline).

## Available Versions

- [JavaScript](./js/)
- [TypeScript](./ts/)
