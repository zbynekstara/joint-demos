# JointJS+: Corporate Organizational Chart <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

Do you want to allow users to define an organizational structure and populate it with people in a drag & drop fashion? Check out this demo leveraging the flexibility of JointJS+ by combining tree and grid layouts and customizing the stencil.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/corporate-organizational-chart).
