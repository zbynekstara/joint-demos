# JointJS+: Selection Alignment <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

We borrowed the arrange and align function from good old PowerPoint. So if you've been racking your brain on how to allow users to select and align objects, you don't have to. Check out the demo below to see how to do it.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/selection-alignment).
