# JointJS+: Dynamic Tooltip Content <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

This demo shows how to display the tooltip when you hover over a port. The tooltip shows various information about the port connection (what it is connected to and how) and the information is gathered on demand from the graph just before the tooltip is displayed.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/dynamic-tooltip-content).

## Available Versions

- [JavaScript](./js/)
