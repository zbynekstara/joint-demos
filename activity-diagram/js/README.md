# JointJS: Activity Diagram (JavaScript)

Have you ever wondered how a bill becomes law in the US? Our activity diagram, a visual representation of workflows of actions, explains this complex process, leveraging some of the JointJS diagramming features like collapsible swimlanes, conditional highlighters or automatic link reconnect.

## How to download this demo

You can download this demo using our [`@joint/cli` tool](https://www.npmjs.com/package/@joint/cli):

```bash
npx @joint/cli download activity-diagram/js
```

Alternatively, you can get the [copy of the repository](https://github.com/clientIO/joint-demos/archive/refs/heads/main.zip) from GitHub as usual.

## Running the application

Install the dependencies by running:

```bash
npm install
```

And then start the application with:

```bash
npm run dev
```
