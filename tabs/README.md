# JointJS+: Tabs <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

The Tabs demo – written in React, Vue, Angular and Svelte – allows the user to display multiple JointJS diagrams while navigating them via a Tabs component.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/tabs).

## Available Versions

- [Angular](./angular/)
- [React](./react/)
- [Vue](./vue/)
- [Svelte](./svelte/)
