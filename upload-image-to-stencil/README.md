# JointJS+: Upload Image to Stencil <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

Do you want users to be able to upload an image directly into the stencil? Drag and drop a stencil element into a diagram and be able to view the original image in the LightBox?

This demo is also available online at [jointjs.com](https://jointjs.com/demos/upload-image-to-stencil).
