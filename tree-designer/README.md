# JointJS+: Tree Designer <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

The Tree Designer application is a graphical tool for creating and modifying tree structures. It enables users to utilize a form-based interface to design and customize tree nodes and edges, as well as add or delete child nodes to adjust the hierarchy as required.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/tree-designer).

## Available Versions

- [JavaScript](./js/)
- [TypeScript](./ts/)
